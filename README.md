# 🌿 Plant Disease Detection

## 📖 Overview
This project identifies plant leaf diseases using a Convolutional Neural Network (CNN) trained on the "Plant Village Disease Classification".

## 🧠 Features
- 38-class classification model
- Real-time prediction via Streamlit web app
- Implemented with TensorFlow & Keras
- Evaluation metrics: Accuracy, Precision, Recall

## 🧩 Technologies
Python, TensorFlow/Keras, Streamlit, OpenCV, Matplotlib, Google Colab

## 📂 Files
- `final.ipynb` – Model training and evaluation
- `Plant_Diseases.keras` – Saved model (linked via Google Drive)
- `app.py` – Streamlit web app for prediction
